## 一、T+0 机制与做空

### T+0 回转
- 当日买入可当日卖出，无次数限制
- 资金交收 T+2

### 做空（融券）
- **标的范围**：港交所指定（2025 约 768 只股 + 352 只基金），多为蓝筹
- **报升原则（Tick Rule）**：卖空价 ≥ 最佳卖盘价，防止打压
- **禁止裸卖空**：必须先借券，否则构成刑事犯罪
- **港股通限制**：内地投资者通过港股通**无法**直接融券，需香港本地券商账户

### 卖空比例上限与监控
- 单只港股通标的 **担保卖空比例 ≤ 20%**
- 达到上限后暂停卖空申报，直至降至 **≤ 18%** 恢复
- 计算：`当日卖空成交额 / 总成交额`，只算建仓不算平仓

### 对选股的含义

| 卖空比例 | 含义 | skill 行动 |
|---------|------|------------|
| < 5% | 正常 | 无操作 |
| 5-10% | 部分看空 | 标注"观察" |
| > 10% | 空头集中 | 硬否决候选 |
| 接近 20% | 严重分歧 + 潜在 Short Squeeze | 硬否决 |

## 二、老千股八项识别清单

| 特征 | 阈值 |
|------|------|
| 1. 股价长期低于 1 HKD（仙股） | 收盘价 < 1 HKD 超 60 日 |
| 2. 频繁供股 + 合股组合 | 5 年内 ≥ 2 次供股 或 ≥ 2 次合股 |
| 3. 大股东高质押 + 低流通 | 质押率 > 60% + 公众持股 < 25% |
| 4. 基本面差 | 连续 3 年净利润 < 0 或商誉/净资产 > 50% |
| 5. 成交量异常 | 平时日成交 < 100 万 HKD，偶有单日爆量 |
| 6. 频繁改名/转型 | 3 年内改名或主营变更 ≥ 2 次 |
| 7. 频繁换核数师 | 3 年内 ≥ 2 次更换审计机构 |
| 8. 关联方大比例配售 | 向关联人低价配新股 |

**任一命中即硬否决**。

## 三、南下资金（港股通）选股逻辑

### 2025 规模与趋势

- 上半年南向资金净流入 > 6900 亿 HKD（接近 2024 全年）
- 截至 11 月累计净买入破 5 万亿 HKD
- 机构预测未来 5 年空间 10 万亿人民币

### 2025 南向资金行业配置

| 行业 | 占比/变化 | 核心逻辑 |
|------|---------|------|
| 非必需消费 | 49-56%（大幅提升） | 新消费、即时零售、国潮 |
| 资讯科技业 | 14-32% | AI、半导体、互联网盈利修复 |
| 金融/能源（高股息） | ~25% | 险资稳定分红偏好 |
| 创新药/医疗 | 外资增持、南向 Q4 减持 | 基本面改善但分歧 |

### 阶段性风格切换

- **上半年进攻**：腾讯、阿里、快手、小米、恒生科技 ETF
- **Q4 转守**：切换到高股息（中海油、工行、中移动）

### 对选股 skill 的启示

- **哑铃策略**：进攻端（科技/新消费）+ 防守端（高股息/金融）
- **动态跟踪**：南下资金月度行业配置变化比持仓绝对值更有信号
- **定价权转移**：南向占比高的港股，定价逻辑"港股 A 股化"（更看 A 股资金面）

## 四、A/HK 市场机制对比总表

| 维度 | A 股 | 港股 |
|------|------|------|
| 交易制度 | T+1 | T+0 |
| 涨跌幅 | ±10% / ±20% / ±30%（分板块）| 无限制 |
| 做空 | 融资融券（个人门槛 50 万） | 公开融券（港股通不支持） |
| 退市 | 面值退 / 市值退 / 财务退 | 停牌 18 个月 / 拒绝复核 |
| 新股首日 | 主板 +44% / 双创无限制 | 无涨跌幅限制 |
| 交收 | T+1 | T+2 |
| 计价货币 | CNY | HKD（部分双柜台 HKD/CNY） |
| 境外投资者准入 | 合格境外（QFII/RQFII）/ 沪深港通 | 开放（需 SFC 持牌券商） |
| 做空比例公开 | 仅融券余额 | 每日公布卖空成交 |

## 五、为什么不能把美股套路直接搬到 A/HK

1. **中国宏观政策主导** > 基本面：政策市 / 资金市压倒业绩
2. **北向 / 南下"聪明钱"**：美股没有这种单一方向跨境资金流的信号
3. **涨跌幅限制 + T+1**（A 股）：美股动量因子计算失效
4. **老千股 / 仙股**（港股）：美股 SEC 监管更严，没这么多"僵尸股"
5. **政策催化**：央行、证监会、经济会议的周期性影响，远强于美国
6. **行业分类差异**：Fama-French 用 GICS，中国用申万/中信/恒生

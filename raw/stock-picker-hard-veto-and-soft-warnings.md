## 一、硬否决清单（A 股）—— 命中任一立即剔除

### 财务健康

| 规则 | 阈值 | 数据字段 | AkShare 接口 |
|------|------|---------|-------------|
| ST/*ST | 名称含 "ST" "*ST" "退" | 股票名称 | `stock_zh_a_spot_em` |
| 面值退市 | 连续 20 日收盘 < 1 元 | close | `stock_zh_a_hist` |
| 市值退市（2024 新规）| 连续 20 日总市值 < 5 亿 | total_mv | `stock_zh_a_spot_em` |
| 商誉 / 净资产 | > 30% 高风险，> 50% 硬否决 | 商誉、净资产 | `stock_sy_em` |
| 大股东质押率 | > 80% | 质押比例 | `stock_gpzy_pledge_ratio_em` |
| 财务造假退市 | 一年造假 ≥ 2 亿且 > 30% | 证监会公告 | `stock_notice_report` |
| 净利润/经营现金流 | < 0 连续 2 年 | 现金流/净利润 | `stock_financial_*` |
| 应收账款/营收 | > 50% 且增速 > 营收增速 | AR/Revenue | `stock_financial_analysis_indicator` |
| 非标审计意见 | 非"标准无保留意见" | 审计意见类型 | `stock_notice_report` |
| 证监会立案调查 | 近 24 个月 | 监管公告 | `stock_notice_report` |

### 行为异常

| 规则 | 阈值 |
|------|------|
| 控股股东减持 | 12 个月累计 > 总股本 5% |
| 审计师更换 | 3 年内 ≥ 2 次 |
| 主营业务变更 | 3 年内 ≥ 2 次 |
| 业绩承诺未完成 | 子公司 ≥ 2 年未达 80% 承诺 |

### 高商誉行业（需加倍警惕）

- **传媒**：游戏/影视监管波动大，历史暴雷重灾区
- **医药生物**：带量采购挤压利润，创新药回报周期长
- **电子/计算机**：消费电子疲软，半导体周期下行
- **教育**："双减"政策冲击
- **机械设备**：并购整合难度大

## 三、Beneish M-Score 模型（财务造假量化识别）

### 公式

```
M = −4.84
    + 0.92 × DSRI     (应收账款指数)
    + 0.528 × GMI     (毛利率指数，恶化信号)
    + 0.404 × AQI     (资产质量指数)
    + 0.892 × SGI     (销售增长指数)
    + 0.115 × DEPI    (折旧率指数)
    − 0.172 × SGAI    (销管费用指数)
    + 4.679 × TATA    (总应计项/总资产, 权重最高)
    − 0.327 × LVGI    (杠杆指数)
```

### 判断标准

| M 值 | 结论 |
|------|------|
| > −1.78 | 存在操纵可能（原版阈值） |
| ≤ −2.22 | 造假可能性低 |
| A 股建议调整为 > −1.89 ~ −2.0 | 本土化阈值 |

### 八变量含义

| 变量 | 含义 |
|------|------|
| DSRI | 应收账款 / 收入 的同比变化 → 收入提前确认信号 |
| GMI | 毛利率恶化 |
| AQI | 非流动资产（非 PP&E）占比变化 → 费用资本化 |
| SGI | 收入增长压力 |
| DEPI | 折旧政策放缓 |
| SGAI | 销管费用变化 |
| **TATA** | **总应计项目/总资产**（对 A 股最敏感） |
| LVGI | 杠杆变化 |

### A 股适用性

- **适用**：对被证监会处罚或 ST 的公司有识别能力；TATA 最敏感
- **局限**：中国会计准则差异 + 关联方交易 + 资金占用 + 体外循环不反映在 8 指标中
- **建议**：
  - 不单独使用，配合 Altman Z-score、Ohlson O-score、现金流匹配度、审计意见
  - 行业内横向比较 M-score 而非绝对阈值
  - 重点看 TATA、DSRI、SGI

## 四、价值陷阱识别（非硬否决，但强警告）

| 特征 | 阈值 |
|------|------|
| ROE 持续下滑 | 连续 3 年下滑 |
| 营收 / 净利润同比 | 连续 2 年负增长 |
| 低 PE 但行业衰退 | PE < 10 且行业 PE 也在下降 |
| 周期股高点低 PE | 周期顶点时 PE 反而最低 |
| 现金流负 + 负债上升 | 双信号同时出现 |

**典型案例**：视源股份 ROE 2019 年 38.38% → 2025 年 7.65%（6 年连跌）。

## 七、skill 输出时的"排雷提示"文案

每只推荐股需包含"潜在风险"字段（与亮点并列）：

```
宁德时代（300750）
  ★ 得分 87/100
  ○ 亮点：ROE 连续 5 年 > 20%，新能源龙头...
  ⚠ 风险：
    - 估值分位 80%（需择时）
    - 商誉/净资产 8%（低风险但需监控）
    - 海外市场政策不确定性
  > 本分析仅供学习参考，非投资建议。
```

新手只看亮点容易追高，**风险字段必须与亮点同等显眼**。

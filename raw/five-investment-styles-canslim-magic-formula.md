## 一、五派速览

| 流派 | 核心逻辑 | 关键指标 | 适用周期 | 代表人物 |
|------|---------|---------|---------|---------|
| **价值** | 买低于内在价值 | 低 PE / PB / 高安全边际 | 长线 | Graham、早期巴菲特、Klarman |

## 二、Greenblatt 神奇公式（Magic Formula）

### 双指标

1. **ROC（资本回报率）** = `EBIT / (净固定资产 + 营运资本)` — 好公司
2. **盈利收益率** = `EBIT / EV`，`EV = 市值 + 有息负债 − 现金` — 便宜价格

### 机械流程

1. 剔除金融、公用事业、市值过小
2. 按 ROC 排名 A（高→低）
3. 按盈利收益率排名 B
4. **综合排名 = A + B**（小者优）
5. 选前 20-30 构组合
6. 持有 1 年后滚动

### 业绩（1988-2004 美股）

- 年化 30.8%，远超标普 500 同期 ~12%

### A 股适用性

- 需剔除 ST、次新股
- 回测依然有效但波动大
- 工具：理杏仁、同花顺 iFinD

## 三、CANSLIM（欧奈尔）—— 成长+动量融合派

### 七要素

| 字母 | 含义 | 核心阈值 |
|------|------|---------|
| **C** | 当季 EPS | 同比 ≥ 20-50%，牛市 40-500% |
| **A** | 年 EPS | 3 年复合增长 ≥ 25-30% |
| **N** | 新产品/新管理/股价新高 | 股价突破 + 放量 |
| **S** | 供需 | 小盘（流通 < 2500 万股），突破时成交量 **>日均 50%** |
| **L** | 领涨 vs 落后 | 相对强度 RPS ≥ 80 |
| **I** | 机构认同 | ≥ 10 家优质机构持仓，警惕过量 |
| **M** | 市场方向 | 顺势；止损铁律 **-7% ~ -8%** |

### 辅助指标（C 的配套）

- 营收同比 ≥ 25%
- ROE > 17%
- 每股现金流/EPS > 20%
- 毛利率/营业利润率逐季提升

## 四、五派陷阱对照

| 流派 | 主要陷阱 | 规避 |
|------|--------|------|
| 价值 | 低 PE 陷阱（ROE 持续下滑 → 戴维斯双杀） | 同时看 ROE 变化 + 行业前景 |
| 成长 | 高估值+业绩变脸 → 戴维斯双杀反向 | PEG < 1 保持纪律 |
| 质量 | ROE 是会计操纵的重灾区 | 看现金流/净利润、扣非净利 |
| 红利 | 经营恶化仍高分红维持股价（挖肉饲虎）| 看派息率 + 分红连续性 + 自由现金流 |
| 动量 | 追高被套（高点同步高波动）| RSI > 90 + 缩量滞涨 → 警告 |

## 五、Fama-French 因子化五派

- **Value** = HML（High Minus Low，book/market）
- **Profitability ≈ Quality** = RMW（Robust Minus Weak）
- **Investment** = CMA（Conservative Minus Aggressive）—— 低资本开支
- **Size** = SMB（Small Minus Big）
- **Market** = MKT

Fama-French 五因子 **没有动量**。Carhart 在 FF3 基础上加了 Mom 形成四因子。

### A 股实证

- 价值策略：银行、煤炭板块有效
- 红利：中证红利指数近年表现亮眼
- 成长：集中在新能源、半导体、AI
- 动量：短期有效，但涨跌停制度扭曲
- 质量：消费、医药板块稳

## 六、Top-down vs Bottom-up

| 路径 | Top-down | Bottom-up |
|------|---------|-----------|
| 思路 | 宏观 → 行业 → 个股 | 个股 → 组合 → 忽略宏观 |
| 适合 | 行业轮动明显的市场（A/HK） | 长期持有优质公司（Buffett） |
| 本项目 | **主路径**（对应用户自然语言查询）| 补充 |
| 风险 | 行业判断错 → 全错 | 个股判断错 → 局部错 |

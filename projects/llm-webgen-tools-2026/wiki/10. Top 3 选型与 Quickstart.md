# Top 3 选型与 Quickstart

基于 8 个维度的分析，针对你的偏好（**a 选型 / 视觉好 / 直接用 / 个人兴趣**），下面是最终推荐的三件套 + 上手路径。

## 推荐三件套

```mermaid
flowchart TB
    Top["三件套战略"]

    Top --> T1["🥇 主力武器<br/>Framer (Mini $5 年付)"]
    T1 --> T1a["视觉天花板 + 直接发布"]
    T1 --> T1b["不打算改代码,接受平台锁定"]

    Top --> T2["🥈 国内零成本方案<br/>Trae SOLO 中国版 (永久免费)"]
    T2 --> T2a["国内直连 + 中文 prompt"]

    Top --> T3["🥉 已有订阅利器<br/>Claude Artifacts"]
    T3 --> T3a["如果你已经付 Claude Pro"]
    T3 --> T3b["复杂 demo / 数据可视化首选"]

    Top --> T4["🎁 按需补充<br/>Readdy / DeepSite V2 / v0"]

    T4 --> T4b["3D / Three.js → DeepSite V2"]

```

## 🥇 Framer Quickstart

### 为什么选它

```mermaid
flowchart TD
    F["Framer Mini 计划"]
    F --> Why["为什么是 #1"]
    Why --> W1["✅ 2026 视觉天花板 (Liquid Glass + Dithered Waves)"]
    Why --> W2["✅ 动画原生一等公民 (Wireframer / Workshop AI)"]
    Why --> W3["✅ 配色 / 字体跟最新趋势"]
    Why --> W4["✅ $5/月 (年付) 比 v0 还便宜"]
    Why --> W5["✅ 国内域可达 (虽然编辑器有时卡)"]

    F --> Tradeoff["接受这个权衡"]
    Tradeoff --> T1["❌ 不能导出 React"]
    Tradeoff --> T2["❌ 平台锁定 (停付费 → 站没了)"]
    Tradeoff --> T3["✓ 你不打算改代码,这就 OK"]
```

### 上手路径（30 分钟）

```mermaid
flowchart TD
    S0["注册 framer.com (Hobby 免费)"]
    S0 --> S1["1) 探索 Marketplace"]
    S1 --> S1a["搜 'Liquid Glass' / 'Dithered Waves'<br/>免费组件加进项目"]

    S1 --> S2["2) 用 Wireframer AI"]
    S2 --> S2a["prompt: 'minimal SaaS landing page,<br/>cyberpunk hero with neon scroll effects'"]
    S2 --> S2b["AI 生成带 scroll-triggered fades + parallax 的初稿"]

    S2 --> S3["3) 用 Workshop AI 加交互"]
    S3 --> S3a["选元素 → '加 hover 3D 旋转'"]
    S3 --> S3b["选标题 → '加 kinetic typography 入场'"]

    S3 --> S4["4) 调色板"]
    S4 --> S4a["跟 2026 趋势:Cloud Dancer 米白 + citrus 撞色<br/>或 Pantone Mocha Mousse 主基"]

    S4 --> S5["5) 决定升级"]
    S5 --> S5a["满意 → Mini 年付 $60 (= $5/月)"]
    S5 --> S5b["买自定义域,移除 watermark"]
```

### Artifact 三类杀手场景

| 场景 | 为什么 Artifact 强 |
|------|-----------------|
| **数据可视化** | SVG / D3.js / Recharts 能直接跑，调色对比度精确 |
| **交互教学 demo** | 单文件 + 实时预览 + 多轮迭代 |
| **复杂概念示意图** | Mermaid 原生，diagram 一次过率高 |
| **算法可视化** | Canvas / SVG 实时演示，搭配 Claude 解释 |

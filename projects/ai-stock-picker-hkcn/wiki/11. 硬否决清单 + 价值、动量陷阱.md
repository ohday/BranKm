# 硬否决清单 + 价值/动量陷阱

选股打分再高，**只要踩到一条硬否决就立即剔除**——这是候选池的第一道门。这一页把 A 股 10 条 + 港股 9 条硬否决规则列成可落地的 checklist（每条带阈值和数据接口），并用 Beneish M-Score 量化财务造假识别。尾部的价值陷阱和动量陷阱是软警告（扣分但不剔除），但新手最容易栽在这两类陷阱里。

## 打分前的过滤漏斗

```mermaid
flowchart TB
    INPUT["候选行业内股票"]

    subgraph STAGE1["Step 1: 硬否决 · 一票否决"]
        H_A["A 股 10 条"]
        H_HK["港股 9 条"]
        H_ALL[/命中任一 → 剔除/]
    end

    subgraph STAGE2["Step 2: 软警告 · 扣分"]
        S_V["价值陷阱"]
        S_M["动量陷阱"]
        S_RES[/每项扣 5-10 分/]
    end

    STAGE3["Step 3: 12 维度正向打分"]

    INPUT --> STAGE1 --> STAGE2 --> STAGE3 --> FINAL["最终排序"]

    H_A & H_HK -.命中.-> REJECT["永久剔除 + 黑名单 6 月"]
    S_V & S_M -.命中.-> REDUCE["扣分但保留"]

    style H_ALL fill:#ff9999
    style S_RES fill:#fff3e0
```

**硬否决和软警告必须分两步**：若合并，"高分股票踩雷"会因为其他维度分高而被错误留在池子里。硬否决是一票否决，不讨价还价。

**港股比 A 股多**的是老千股识别——A 股有强制退市和面值退，港股没有等价机制，所以仙股生态长期存在[^40]。

## Beneish M-Score：量化财务造假识别

```mermaid
flowchart TB
    FIN_ST["财务报表<br/>资产/收入/利润"]

    FIN_ST --> MCALC["M-Score 计算"]

    subgraph FORMULA["公式（8 变量）"]
        F1["−4.84<br/>+ 0.92 DSRI"]
        F2["+ 0.528 GMI<br/>+ 0.404 AQI"]
        F3["+ 0.892 SGI<br/>+ 0.115 DEPI"]
        F4["− 0.172 SGAI<br/>+ 4.679 TATA"]
        F5["− 0.327 LVGI"]
    end

    MCALC --> FORMULA
    FORMULA --> DECISION{"M 值判断"}

    DECISION -->|> -1.78| HIGH["操纵可能高"]
    DECISION -->|-2.22 ~ -1.78| GREY["灰色区"]
    DECISION -->|≤ -2.22| SAFE["低风险"]

    style HIGH fill:#ffcccc
    style SAFE fill:#ccffcc
```

### 八变量含义

| 变量 | 含义 | 作用 |
|------|------|------|
| **DSRI** | 应收账款 / 收入的同比变化 | 收入提前确认信号 |
| **GMI** | 毛利率恶化 | 基本面下行掩饰 |
| **AQI** | 非流动资产（非 PP&E）占比 | 费用资本化 |
| **SGI** | 收入增长压力 | 业绩压力 |
| **DEPI** | 折旧率放缓 | 利润虚增 |
| **SGAI** | 销管费用变化 | 成本隐藏 |
| **TATA** | **总应计项/总资产（权重最高）** | **对 A 股最敏感** |
| **LVGI** | 杠杆变化 | 财务压力 |

### A 股适用性[^45]

- **阈值本土化**：原版 -1.78 → A 股建议 **-1.89 ~ -2.0**
- **TATA 最敏感**：A 股造假公司最常见的异常
- **局限**：中国会计准则差异 + 关联方交易 + 资金占用 + 体外循环不反映在 8 指标中
- **使用建议**：不单独使用，配合 Altman Z-score、Ohlson O-score、现金流/净利润匹配、审计意见类型

### 集成到 skill

```mermaid
flowchart LR
    STOCK["候选股票"]

    STOCK --> MC["计算 M-Score"]
    MC --> CHK{"M > -1.89?"}

    CHK -->|是| ADD_SOFT["加软警告<br/>扣 15 分"]
    CHK -->|是 且 TATA 极端| HARD["升级为硬否决"]
    CHK -->|否| PASS["通过"]
```

## 价值陷阱识别（软警告）

```mermaid
flowchart TD
    VT["价值陷阱"]

    subgraph SIGNAL["五大信号"]
        S1["ROE 连续 3 年下滑<br/>（盈利能力退化）"]
        S2["营收 / 净利润 连 2 年负增长"]
        S3["低 PE 但行业也在衰退<br/>（PE < 10 + 行业 PE 下降）"]
        S4["周期股高点低 PE<br/>（反向解读陷阱）"]
        S5["现金流负 + 负债上升<br/>（双信号）"]
    end

    VT --> SIGNAL

    subgraph RESULT["戴维斯双杀"]
        R1["EPS 下降"] --> R2["PE 下降"] --> R3["股价 = PE × EPS<br/>双重下跌"]
    end

    SIGNAL --> RESULT

    style R3 fill:#ffcccc
```

### 典型案例

**视源股份**：ROE 2019 年 38.38% → 2025 年 7.65%（6 年连跌）[^45]。即使 PE 分位已经很低，基本面持续恶化，股价始终不振。

### skill 检测规则

```python
# 伪代码
def is_value_trap(stock):
    score = 0
    if declining_roe_for_3_years(stock): score += 1
    if negative_growth_for_2_years(stock): score += 1
    if low_pe_in_declining_industry(stock): score += 1
    if cyclical_stock_at_peak_low_pe(stock): score += 1
    if negative_cf_and_rising_debt(stock): score += 1
    return score >= 2  # 命中 2 条以上 → 价值陷阱软警告
```

## 硬否决 vs 软警告的完整 skill 流程

```mermaid
flowchart TB
    INPUT_ALL["候选股票"]

    subgraph STAGE1_PROC["Stage 1: 硬否决"]
        H_CHECK["遍历 A 股 10 条 / 港股 9 条"]
        H_OR{"任一命中?"}
        H_OR -->|是| REMOVE["永久剔除<br/>+ 黑名单 6 月"]
        H_OR -->|否| CONT1["通过 Stage 1"]
    end

    subgraph STAGE2_PROC["Stage 2: 软警告"]
        S_V["价值陷阱检测"]
        S_M["动量陷阱检测"]
        S_MS["M-Score 检测"]
        S_V & S_M & S_MS --> S_SUB["每项扣分"]
    end

    subgraph STAGE3_PROC["Stage 3: 12 维度打分"]
        D_12["加权合成总分"]
    end

    INPUT_ALL --> STAGE1_PROC
    CONT1 --> STAGE2_PROC --> STAGE3_PROC --> OUT["最终分数 + 陷阱警告"]

    style REMOVE fill:#ff9999
    style OUT fill:#ffe8a8
```

## 输出时的"排雷提示"文案

每只推荐股必须含**与亮点同等显眼**的风险字段：

```
宁德时代（300750）
  ★ 得分 87/100

  ○ 亮点
    - ROE 连续 5 年 > 20%
    - 新能源全球龙头（份额 37%）
    - Q2 装机量创新高

  ⚠ 风险（不容忽视）
    - D1 估值：分位 82%（偏贵）
    - 商誉/净资产 8%（低风险但需监控）
    - 海外市场政策不确定性
    - 主要客户集中度 > 40%

  ℹ 本分析仅供学习参考，非投资建议。
```

**新手阅读偏差**：只看亮点容易追高，**风险字段必须与亮点同等显眼**——不要放在页脚小字。

# 五大流派速读

投资世界里「流派」不是标签，是**看世界的坐标系**。价值派眼里的好股票是"便宜"，动量派眼里的好股票是"强势"，这两种人对同一张财报看到的是不同的数字。这一页给你五派的完整对照，以及一个可以立刻用的工具：**根据你对四个场景的反应，反推你属于哪一派**。

## 流派全景图

```mermaid
flowchart TD
    ROOT["投资流派"]

    ROOT --> V["价值派 Value<br/>买低于内在价值<br/>Graham · 早期 Buffett"]
    ROOT --> G["成长派 Growth<br/>为高速增长付溢价<br/>Fisher · Lynch"]
    ROOT --> Q["质量派 Quality<br/>持久竞争优势<br/>中后期 Buffett · Greenblatt"]
    ROOT --> D["红利派 Dividend<br/>稳定现金流回报<br/>Bogle · Dreman"]
    ROOT --> M["动量派 Momentum<br/>强者恒强 趋势跟随<br/>Driehaus · O'Neil"]

    V -.擅长.-> V1["熊市 · 周期股低点 · 金融地产"]
    G -.擅长.-> G1["科技 · 生物医药 · 新兴行业"]
    Q -.擅长.-> Q1["消费 · 医药 · 长期持有"]
    D -.擅长.-> D1["退休资金 · 低利率环境 · 公用事业"]
    M -.擅长.-> M1["牛市 · 趋势明确 · 高流动性"]
```

## 关键指标对照表

每一派看的指标集合几乎不重叠。

| 维度 | 价值 | 成长 | 质量 | 红利 | 动量 |
|------|:---:|:---:|:---:|:---:|:---:|
| PE / PB 分位 | ★★★ | ★ | ★★ | ★★ | - |
| 营收/净利增速 | ★ | ★★★ | ★★ | ★ | ★ |
| ROE / 毛利率 | ★★ | ★★ | ★★★ | ★★ | ★ |
| 市场份额 / 护城河 | ★★ | ★★ | ★★★ | ★ | - |
| 股息率 / 派息率 | ★★ | ★ | ★★ | ★★★ | - |
| 相对强度 RPS | - | ★ | - | - | ★★★ |
| 技术形态突破 | - | ★ | - | - | ★★★ |
| 资金流 / 北向 | ★ | ★★ | ★ | ★ | ★★★ |

★ 个数就是维度权重的直觉映射。`-` 不是"不看"，是"不作为主要决策依据"。

## 两个机械公式流派（可以立即用）

### Greenblatt 神奇公式（Magic Formula）

Joel Greenblatt 的名著《股市稳赚》提出的"好公司 + 便宜"机械选股法[^43]。

```mermaid
flowchart LR
    subgraph INPUT["全市场（剔除金融 · 公用事业 · 小市值）"]
        S1["股票池"]
    end

    subgraph A["步骤 A: 好公司"]
        ROC["ROC = EBIT / 资本占用<br/>= EBIT / (净固定资产 + 营运资本)"]
        ROC --> RA["按 ROC 从高到低排名 A"]
    end

    subgraph B["步骤 B: 便宜"]
        EY["盈利收益率 = EBIT / EV<br/>EV = 市值 + 有息负债 − 现金"]
        EY --> RB["按收益率从高到低排名 B"]
    end

    subgraph C["步骤 C: 合成"]
        SUM["综合排名 = A + B"]
        SUM --> TOP["选前 20-30 只"]
    end

    INPUT --> A
    INPUT --> B
    A --> C
    B --> C
    TOP --> HOLD["持有 1 年 · 滚动再筛"]
```

**为什么用 EBIT 而不是净利润？为什么用 EV 而不是市值？** ——为了消除资本结构（负债高低）和税率差异，让不同公司能横向比较。净利润会被利息支出和税率扭曲。

**历史业绩**（1988-2004 美股，Greenblatt 书中披露）：年化 **30.8%**，远超标普 500 同期 ~12%[^43]。

**A 股适用性**：需剔除 ST、次新股；本土化可调阈值；回测依然有效但波动大。工具：理杏仁、同花顺 iFinD。

### O'Neil CANSLIM（成长+动量融合派）

威廉·欧奈尔在《笑傲股市》中提出的七要素法则，被美国个人投资者协会评为最稳定的选股系统之一[^43]。

```mermaid
flowchart TB
    subgraph BASIC["基本面筛选"]
        C["C: 当季 EPS 同比 ≥ 20-50%<br/>牛市优选 40-500%"]
        A["A: 年 EPS 3 年复合 ≥ 25-30%"]
        N["N: 新产品 · 新管理 · 股价新高"]
    end

    subgraph TECH["技术面筛选"]
        S["S: 突破时放量 > 均量 50%<br/>偏好小盘（流通 < 2500 万股）"]
        L["L: 领涨股 RPS ≥ 80"]
    end

    subgraph CONFIRM["确认"]
        I["I: ≥ 10 家优质机构持仓<br/>警惕过量持有"]
        M["M: 顺势 · 止损 -7~-8%"]
    end

    BASIC --> TECH --> CONFIRM --> BUY["买入"]

    CONFIRM -.失败.-> EXIT["退出观察"]
```

**关键点**：
- 辅助指标——营收 ≥ 25%、ROE > 17%、每股现金流/EPS > 20%
- **止损铁律 -7~-8%**：单次最大亏损不超过买价 8%，这是 O'Neil 体系的核心纪律

## 每派的陷阱（新手最容易栽的地方）

```mermaid
flowchart TD
    subgraph VT["价值陷阱"]
        VT1["PE 低 → 看起来便宜"]
        VT2["ROE 连续下滑"]
        VT3["行业衰退 / 产能过剩"]
        VT4["戴维斯双杀<br/>EPS 下降 + PE 下降"]
        VT1 --> VT2 --> VT3 --> VT4
    end

    subgraph GT["成长陷阱"]
        GT1["高 PE 容忍"]
        GT2["业绩一次变脸"]
        GT3["戴维斯双杀（反向）<br/>EPS 打折 + PE 崩塌"]
        GT1 --> GT2 --> GT3
    end

    subgraph MT["动量陷阱"]
        MT1["强势股 RSI > 90"]
        MT2["缩量滞涨"]
        MT3["回调 30%+"]
        MT1 --> MT2 --> MT3
    end

    VT4 -.代表.-> CASE_V["视源股份<br/>ROE 2019: 38.38% → 2025: 7.65%"]
    GT3 -.代表.-> CASE_G["高景气突然熄火的成长股"]
    MT3 -.代表.-> CASE_M["跟风追高被套"]
```

**戴维斯双杀**（Davis Double Kill）是选股最凶险的反模式：EPS 下降 → PE 下降 → 股价 = PE × EPS 双重下跌。反过来"戴维斯双击"（EPS 上升 + PE 上升）是成长股的理想路径[^45]。

## Fama-French 因子模型：流派的学术化

学术界把流派"因子化"：Fama 教授等人在 2015 年五因子模型中确认了**价值**（HML）、**质量**（RMW）、**投资保守度**（CMA，相当于反向成长）、**规模**（SMB）、**市场**（MKT）五个因子——**但没有动量**[^43]。

```mermaid
flowchart LR
    FF3["Fama-French 三因子 (1993)<br/>Market + Size + Value"]
    FF5["Fama-French 五因子 (2015)<br/>+ Profitability + Investment"]
    CARHART["Carhart 四因子 (1997)<br/>Fama 三因子 + Momentum"]

    FF3 --> FF5
    FF3 --> CARHART

    FF5 -.对标.-> STYLES1["价值 + 质量 + (反向)成长 + 规模"]
    CARHART -.对标.-> STYLES2["价值 + 规模 + 动量"]
```

**为什么五因子不含动量**？Fama 认为动量缺乏基于风险的合理解释，且套利成本高。但实证显示动量因子独立存在（Carhart 模型），因此后来者多在 FF5 基础上叠加动量。

## A 股各派有效性（实证结论）

- **价值**：银行、煤炭板块有效
- **红利**：中证红利指数近年跑赢大盘
- **成长**：集中在新能源、半导体、AI
- **动量**：短期有效，但涨跌停制度扭曲
- **质量**：消费、医药板块长期稳定

详细的 A 股八大因子体系与 IR 加权方法见 [5. 个股 12 维度体系](5.%20个股%2012%20维度体系%20%2B%20周期权重.md)。

## 流派自测（Socratic 题源）

skill 的引导问卷不会直接问你"你是哪一派"（新手答不出），而是**通过情景反推**。这里给你四道最核心的判别题[^47]：

```mermaid
flowchart TD
    Q1["你的股票从 100 → 130 → 110<br/>最强的感受？"]
    Q1 -->|懊恼 130 没卖| TYPE_ANCHOR["锚定效应强 → 偏价值"]
    Q1 -->|感觉还赚着| TYPE_LONG["长线思维 → 偏质量/红利"]

    Q2["下列哪只股票你最想买？<br/>A: PE 10 · ROE 12% · 股息 5% · 5 年零增长<br/>B: PE 40 · ROE 25% · 股息 0 · 3 年营收翻 3 倍<br/>C: PE 25 · ROE 20% · 股息 2% · 复合增长 20%"]
    Q2 -->|选 A| PICK_A["价值 / 红利"]
    Q2 -->|选 B| PICK_B["成长"]
    Q2 -->|选 C| PICK_C["质量"]

    Q3["热门股没买 涨了 3 倍<br/>你会？"]
    Q3 -->|立刻追| FOMO["动量倾向"]
    Q3 -->|等回调| DISCIPLINE["纪律性强 → 偏价值/质量"]
    Q3 -->|不追| IGNORE["独立思考 → 偏质量"]

    Q4["你的股票亏了 20%<br/>你会？"]
    Q4 -->|立刻卖| RISK_LOW["风险厌恶高 → 偏红利/质量"]
    Q4 -->|分析再加仓| CONVICTION["判断强 → 偏价值/成长"]
    Q4 -->|毫不动摇| DEEP["深度研究派 → 偏质量"]
```

**skill 最终推断出的不是单一派别**，而是一个权重向量：`{value: 0.6, growth: 0.7, quality: 0.8, dividend: 0.3, momentum: 0.4}`，让不同维度按此权重组合激活（细节见 [9. Socratic 引导问卷](9.%20Socratic%20引导问卷.md)）。

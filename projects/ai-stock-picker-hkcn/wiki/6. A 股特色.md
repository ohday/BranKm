# A 股特色

A 股不是美股的廉价版——它有**自己的交易制度、资金信号和退市规则**。直接搬美股选股逻辑到 A 股会栽在涨跌停板归一化、ST 识别、北向"聪明钱"这几道坎上。这一页把你必须知道的 A 股特殊机制梳理成一张地图，并给出在选股中怎么用它们。

## A 股六大独有机制

```mermaid
flowchart TD
    ROOT["A 股独有的选股要素"]

    ROOT --> M1["① 涨跌停板<br/>按板块差异"]
    ROOT --> M2["② ST / *ST / 退市"]
    ROOT --> M3["③ 北向资金<br/>陆股通聪明钱"]
    ROOT --> M4["④ 龙虎榜<br/>资金席位公开"]
    ROOT --> M5["⑤ 融资融券<br/>余额变化信号"]
    ROOT --> M6["⑥ 注册制分层<br/>主板/双创/北交所"]
```

每一项都**改变了因子的意义或有效性**。下面逐一展开。

## 机制 1：涨跌停板（按板块分层）

```mermaid
flowchart LR
    subgraph LIMIT["涨跌幅限制"]
        MAIN["主板沪/深<br/>±10%<br/>ST ±5%"]
        GROWTH["创业板/科创板<br/>±20%<br/>ST ±20%"]
        BSE["北交所<br/>±30%<br/>ST ±30%"]
    end

    subgraph IPO["新股上市"]
        I1["主板首日 +44%"]
        I2["双创前 5 日无限制"]
        I3["北交所首日无限制"]
    end

    LIMIT & IPO -.影响.-> CONSEQUENCE["同样一个『涨停』<br/>不同板块含义完全不同"]
```

### 对选股因子的意义

```mermaid
flowchart TB
    SAME["同样：『连续 3 个涨停』"]

    SAME --> P1["主板 +33.1%<br/>（强势但非暴涨）"]
    SAME --> P2["创业板 +72.8%<br/>（接近翻倍）"]
    SAME --> P3["北交所 +119.7%<br/>（翻倍还多）"]

    P1 & P2 & P3 --> NEED["skill 必须做<br/>板块归一化"]
```

**归一化规则**[^39]：

| 因子 | 主板 | 创业板/科创板 | 北交所 |
|------|:---:|:---:|:---:|
| "涨停"阈值 | +9.8%~10% | +19.8%~20% | +29.8%~30% |
| "连板"意义 | 强 | 中（涨停跌停都易） | 弱（波动大） |
| "光头阳线"实体 | > 5% | > 10% | > 15% |

否则跨板块比较会得到"北交所股票动量最强"这种错误结论。

### 2024 退市新规关键变化[^39]

```mermaid
        C1["市值退市<br/>3 亿 → 5 亿"]

        C3["分红不达标<br/>新增 → ST"]

    end

    NEW --> CHANGES
```

### 面值退 / 市值退阈值

- **面值退**：连续 **20 日** 收盘价 **< 1 元**
- **市值退**：连续 **20 日** 总市值 **< 5 亿**

**skill 的预警红线**：当股价连续 10 日低于 1.5 元，或市值连续 10 日低于 6 亿，就要提前加入硬否决。不要等真正触发退市才剔除——等那时候再卖已来不及[^45]。

### 2026 Q1 末真实数据案例[^40]

| 行业 | 持仓市值 | 占比 | 核心标的 |
|------|---------|------|---------|
| **电力设备（新能源）** | 5120 亿元 | ~20% | 宁德时代（3058 亿元） |
| **电子（半导体）** | 3749 亿元 | 14.5% | 中际旭创、北方华创 |
| **有色金属** | 1746 亿元 | 6.7% | 紫金矿业、天齐锂业 |

**加仓方向**：AI 算力（光模块"三杰"单季加仓 230+ 亿）、商业航天、存储芯片、可控核聚变。

**减仓方向**：消费电子（-321 亿）、非银金融（-314 亿）、汽车、传统银行。

### 使用北向数据的两个陷阱

1. **中间通道污染**：并非所有"北向资金"都是真外资——部分是内地通过香港通道的资金，有些是托管通道反向买卖。**绝对值不可靠，变化率更有效**。
2. **数据延迟**：北向数据每日更新，但机构行业分析要等**季报披露**。skill 用日频北向做行业判断时，要与月度趋势对照，避免单日噪声。

## 机制 4：龙虎榜（A 股特色的资金席位公开制度）

```mermaid
flowchart TD
    TRIGGER["触发上榜条件<br/>日涨跌幅偏离 > 7% 等"]

    TRIGGER --> LIST["交易所公布<br/>当日买卖前 5 席位"]

    LIST --> INST["机构席位<br/>公募 / 保险 / 社保"]
    LIST --> HOT["知名游资<br/>拉萨东财 / 华鑫上海 / 中信杭州延安路"]

    INST --> READ1["机构净买入<br/>→ 基本面看好"]
    HOT --> READ2["游资打板<br/>→ 短期题材"]

    READ1 & READ2 --> WARN["陷阱：<br/>一日游（次日出货）<br/>对倒（虚假交易量）"]
```

### 对 skill 的使用策略

- **机构净买入 + 龙虎榜上榜** → D8 资金流加分
- **连续 3 日游资打板 + 净卖出机构** → 警示标记（短期过热）
- **跟买知名游资** → **skill 默认不建议**（游资策略快进快出，散户跟买常套在高位）

## 机制 5：融资融券余额（A 股杠杆信号）

```mermaid
flowchart LR
    BALANCE["融资余额"]

    BALANCE --> UP["持续上升<br/>→ 加杠杆看多"]
    BALANCE --> DOWN["持续下降<br/>→ 减杠杆避险"]

    subgraph CROSS["与股价交叉验证"]
        UP & PRICE_UP["股价上涨"] -.正常.-> HEALTHY["上涨有杠杆支撑"]
        UP & PRICE_DN["股价下跌"] -.警告.-> DANGER["融资盘被套 强平风险"]
    end
```

**A 股个人融资融券门槛**：50 万资产 + 20 日日均，远高于美股的"Level 2"。所以融资余额本质上代表**成熟投资者 + 大户**的态度——比散户更有信息优势。

## 机制 6：注册制分层

```mermaid
flowchart LR
    subgraph BOARDS["A 股四板块"]
        MAIN["主板（沪深）<br/>传统 + 核准制"]
        KECHUANG["科创板<br/>硬科技 · 注册制"]
        CHUANG["创业板<br/>成长型 · 注册制"]
        BSE["北交所<br/>专精特新 · 注册制"]
    end

    subgraph RULE_MAP["规则映射"]
        R1["涨跌幅 ±10 / ±20 / ±30"]
        R2["上市条件 盈利 → 经营能力"]
        R3["退市效率 提升"]
    end

    BOARDS --> RULE_MAP
```

**2023-02-17 全面注册制**之后，上市条件从"持续盈利"改为"持续经营能力"——意味着**科创板/创业板有未盈利上市**的公司。这对传统 PE 估值完全失效，必须用 PS/DCF 或行业相对估值。

**skill 的应对**：对注册制板块（尤其未盈利股票），自动切换估值方法到 PS 或"用户只看质量+成长+资金流"。

## AH 两地上市与 AH 溢价

```mermaid
flowchart TB
    AH["同一家公司<br/>A 股 + 港股上市"]

    AH --> COMP["AH 溢价指数"]
    COMP --> HIGH["A 股溢价 > 50%<br/>（常见：工商银行、中信证券）"]
    COMP --> LOW["A 股溢价 < 0%<br/>（罕见：少数港股贵）"]

    HIGH -.选股策略.-> STRATEGY["A 股贵 → 港股便宜<br/>长期套利机会"]
```

**不要天真套利**：AH 溢价长期存在的原因是**流动性溢价 + 制度差异 + 境内人民币稀缺**，不会简单回归。但当溢价达到历史 95% 分位时，做多港股 H 是统计上合理的操作。

```
  龙虎榜上榜: 近 30 日 0 次
```

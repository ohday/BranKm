# 港股特色

港股不是"无涨跌幅的 A 股"。**T+0 回转、公开融券、南下聪明钱、老千股生态、同股不同权**——每一项都彻底改变了选股逻辑。2025 年南下资金突破 5 万亿港元后，港股定价权正从外资向内地资金转移，但老千股生态依旧存在，**对新手是雷区**。这一页把港股独有的五件事讲清楚，以及 skill 怎么用它们。


## 机制 1：T+0 + 无涨跌幅限制

```mermaid
flowchart LR
    subgraph A_VS_HK["A 股 vs 港股"]
        A["A 股<br/>T+1<br/>±10% / ±20%"]
        HK_M["港股<br/>T+0 回转<br/>无涨跌幅"]
    end

    HK_M --> EFFECT1["单日可暴涨暴跌<br/>（+50% / -50% 常见）"]
    HK_M --> EFFECT2["当日买入可当日卖出<br/>无次数限制"]

    EFFECT1 -.对选股.-> IMPL1["不能照搬 A 股<br/>的动量阈值"]
    EFFECT2 -.对选股.-> IMPL2["高波动要求<br/>更高风险警示"]
```

**交收周期**：虽然交易是 T+0，但资金与股票**实际交收在 T+2**。

### 对 skill 动量因子的意义

A 股的 "连 3 板"（+33.1%）在港股可能**一天就完成**——动量因子的时间窗口要缩短。**skill 对港股自动把 D5 动量的计算窗口从 20 日缩到 10 日，MA 从 60/20 改为 30/10**。

### 卖空比例：选股的强信号

```mermaid
flowchart TD
    SSR["单日卖空成交额 / 总成交额<br/>（只算建仓不算平仓）"]

    SSR --> LEVEL{"卖空比例"}

    LEVEL -->|< 5%| NORMAL["正常<br/>skill 无动作"]
    LEVEL -->|5-10%| WATCH["部分看空<br/>标注观察"]
    LEVEL -->|> 10%| VETO["硬否决候选"]
    LEVEL -->|接近 20%| EXTREME["严重分歧<br/>+ 潜在 Short Squeeze<br/>硬否决"]

    EXTREME -.监管.-> LIMIT["达到 20% 后<br/>暂停卖空申报<br/>直至 ≤ 18% 恢复"]

    style VETO fill:#ffcccc
    style EXTREME fill:#ff9999
```

**为什么 > 10% 就硬否决**？卖空集中代表**专业机构判断该股将下跌**——外资机构做空的信息优势通常高于散户，跟做空比跟做多更有效。Short Squeeze 虽然可能带来暴涨，但那是博弈行为不是选股[^40]。

### 对 skill 的使用方式

| 时间尺度 | 数据 | 解读 |
|---------|------|------|
| **日度** | 南下净买入总额 | 市场情绪 |
| **周度** | 行业增持排名变化 | 轮动方向 |
| **月度** | 个股持股变化 | 长线判断 |
| **季度** | 前十大股东 | 定价权转移 |

**"港股 A 股化"趋势**：南下占比高的股票（腾讯、美团、小米），定价逻辑开始跟着 A 股资金面走——不再完全是外资主导。这对 skill 是好消息：**A 股和港股的估值逻辑正在收敛**。

### 老千股八项识别清单[^40]

```mermaid
flowchart TB
    CHECK["8 项硬否决规则"]

    CHECK --> C1["① 股价 < 1 HKD 超 60 日（仙股）"]
    CHECK --> C2["② 5 年内 ≥ 2 次供股或合股"]
    CHECK --> C3["③ 大股东质押 > 60% + 公众持股 < 25%"]
    CHECK --> C4["④ 连 3 年亏损 或 商誉/净资产 > 50%"]
    CHECK --> C5["⑤ 平时日成交 < 100 万 HKD"]
    CHECK --> C6["⑥ 3 年内 ≥ 2 次改名 / 主营变更"]
    CHECK --> C7["⑦ 3 年内 ≥ 2 次更换审计师"]
    CHECK --> C8["⑧ 关联方大比例低价配售"]

    C1 & C2 & C3 & C4 & C5 & C6 & C7 & C8 --> RESULT["任一命中<br/>硬否决"]

    style RESULT fill:#ff9999
```

**为什么港股老千股比 A 股多**？
1. 港交所上市门槛低（历史原因）
2. 无面值退市制度
3. 港交所过去对供股、合股监管宽松（2018 年后收紧但存量仍多）
4. 做空和操纵工具多，小盘股容易被玩坏

### 老千股生态的 skill 设计

```mermaid
flowchart LR
    HK_LIST["全港股 ~2600 只"]

    HK_LIST --> FILTER["老千股过滤器<br/>8 项硬否决"]
    FILTER --> SAFE["安全池 ~1500 只"]
    FILTER --> DANGER["危险池 ~1100 只<br/>（含老千 + 僵尸股）"]

    SAFE --> SCAN["12 维打分"]
    DANGER --> BLACKLIST["永久屏蔽<br/>除非用户手动解除"]

    style DANGER fill:#ffcccc
    style BLACKLIST fill:#ff9999
```

**新手建议**：除非你有极强的研究能力，**永远不要碰港股仙股**。宁可错过 10 个妖股机会，也不要踩一次老千。

## 机制 5：同股不同权 + AH 双柜台

```mermaid
flowchart TB
    subgraph WS["同股不同权（W 股）"]
        W1["阿里巴巴-SW · 小米-W · 美团-W"]
        W1 --> INVESTOR["一股一票 → 不平等投票权<br/>创始人保持控制"]
        W1 --> RISK_W["小股东对公司决策<br/>话语权有限"]
    end

    subgraph DUAL["双柜台（HKD/CNY）"]
        D1["港币柜台（传统）"]
        D2["人民币柜台（新增）"]
        D1 & D2 --> BENEFIT["内地投资者<br/>可直接用 CNY 交易"]
    end
```

### 对 skill 的意义

- **W 股是港股科技主力**：skill 不应因"同股不同权"而扣分，否则会错过腾讯以外几乎所有互联网龙头
- **但治理风险要提示**：创始人离任、管理层内斗时，小股东维权渠道有限
- **AH 双柜台套利**：理论上 HKD/CNY 柜台价格会因汇率偏离短暂失衡，但对新手 skill 不建议引导套利

## AH 溢价

```mermaid
flowchart LR
    AHP["恒生 AH 股溢价指数<br/>= A 股加权价 / H 股加权价"]

    AHP --> HIGH["> 130<br/>A 股贵 30%+<br/>（常态）"]
    AHP --> NORMAL["100-130<br/>正常区间"]
    AHP --> LOW["< 100<br/>港股贵<br/>（罕见）"]

    HIGH --> STRATEGY["历史分位 95%+<br/>→ 做多港股 H 股<br/>统计套利"]
```

**为什么 A 股长期溢价于 H 股**？
1. 流动性溢价（A 股流通盘更活跃）
2. 境内人民币资产稀缺
3. 散户占比高推高估值
4. 资本管制（QDII 额度有限）

**长期回归的可能性低**——这是制度性差异，不是错误定价。但**极端点可以操作**。

## skill 对港股的特殊流程

```mermaid
flowchart TD
    Q_HK["用户查询涉及港股"]

    Q_HK --> S1["Step 1: 老千股过滤<br/>8 项硬否决"]
    S1 --> S2["Step 2: 仙股 + 低流动性剔除<br/>股价 < 1 HKD 或市值 < 5 亿 HKD"]
    S2 --> S3["Step 3: 卖空比例检查<br/>> 10% 排除"]
    S3 --> S4["Step 4: 6-7-5 行业分类<br/>恒生 12 一级 vs A 股"]
    S4 --> S5["Step 5: 南下资金行业增持"]
    S5 --> S6["Step 6: 12 维打分"]
    S6 --> OUT["输出候选"]
```

注意 **前三步是港股专属的过滤**，比 A 股多做的工作——这是港股生态复杂性的代价。
